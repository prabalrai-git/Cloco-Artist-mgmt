# To Run Backend Project

## System Requirements

1. Node.js installed
2. MySQL server running

## Setup Instructions

1. Navigate to the backend directory:
   -> cd backend

2. Install Dependencies:
   -> npm install

3. Seed the database:
   -> node src/config/seedDB.js

4. Run the server:
   -> npm run dev

# To Run Frontend Project

## System Requirements

## Setup Instructions
